## After the placement runs

**Track in a sheet:**
- Sent date
- Open rate (they should report)
- Click-throughs (Vercel Analytics + UTM filter)
- Waitlist signups within 7 days
- $/signup unit economics

**Decide on continuation:**
- If <50 signups per $5k → stop. Wrong audience.
- If 50–150 → tweak (try a different newsletter day, different subject line, different CTA)
- If 150+ → lock 4-week recurring + start outreach to the next-tier newsletters (Exec Sum, Stocktwits Daily)
